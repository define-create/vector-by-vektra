Notes on this mockup

  • Banner uses a muted amber accent (bg-amber-500/10 + border-amber-500/30) — distinct from existing emerald (momentum) and rose (loss) cues, but still neutral / informational, not error-like. Open for review per PRD Open Question #3.
  • Cards under preview render with normal styling — no dimming, badges, or overlays. The banner alone is the preview cue (PRD 4.3.10).
  • Demo data shown reflects PRD 4.2.6 intent: realistic mixed trajectory (6+ matches, W/L mix, mild upward trend, rating in 1000–1100 range). Concrete values are illustrative — left to implementer per Open Question #2.
  • Frame ② "After Got it" is what the same user would see today, pre-feature. It illustrates exactly what the preview is replacing.
  • Frame ③ "Auto-graduated" simulates a user just past the threshold (5 real matches). Note thin trajectory (5 points) and modest values — this is real data, no demo content.
  • Not pictured: the same banner appears on other personal-stats screens per PRD 4.6 (list to be confirmed during implementation).
